Установка Git

В этой книге рассматривается только работа с Git из командной строки. Главным образом потому, что это единственный способ выполнения всех Git-команд, — в большинстве GUI для простоты реализованы только некоторые варианты функциональности Git. Умеющий работать с Git из командной строки сможет разобраться, что делать с GUI-версией. Для пользователей Windows Git for Windows. Инструкции по установке в других операционных системах семейства Unix вы найдете на сайте Git Git for Unix.

Проверяем доступность Git на Вашем компьютере

$ git --version

Если Git установлен, то получим ответ:

git version 2.30.0.windows.2

версия может быть другая

Первая настройка

При установке Git первым делом следует указать имя пользователя и адрес электронной почты.

Ваш идентификатор(имя):

$ git config --global user.name "Sergio Ostanioni"

Ваша электронная почта:

$ git config --global user.email ostanioni@gmail.com

Выбор редактора

$ git config --global core.editor emacs для редактора Emacs

Если Вы используете другой редактор, то выберите команду

Передача параметра --global позволяет сделать эти настройки всего один раз, так как в этом случае Git будет использовать данную информацию для всех ваших действий в системе. Если для конкретного проекта требуется указать другое имя или адрес электронной почты, войдите в папку с проектом и выполните эту команду без параметра --global.

Проверка настроек

Проверить выбранные настройки позволяет команда git config --list $ git config --list user.name=John Doe user.email=johndoe@example.com color.status=auto color.branch=auto color.interactive=auto color.diff=auto ...
